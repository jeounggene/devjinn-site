## Deployment

This site is ready to deploy on [Vercel](https://vercel.com/):
1. Push your code to a GitHub/GitLab/Bitbucket repo.
2. Import the repo into Vercel.
3. Vercel will auto-detect the Vite build and deploy.

## Local Development

```bash
npm install
npm run dev
```

## Build for Production

```bash
npm run build
```
